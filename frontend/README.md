# InsightIQ Frontend

This is the Next.js frontend for the InsightIQ AI platform.

## 🚀 Getting Started

For full project documentation, including setup, architecture, and testing instructions, please refer to the **[Main README](../README.md)** in the root directory.

### Local Development

1. **Install dependencies**:
   ```bash
   npm install
   ```

2. **Run the development server**:
   ```bash
   npm run dev
   ```

3. **Run tests**:
   ```bash
   npm test
   ```

## 🏗️ Tech Stack
- Next.js 15
- TypeScript
- Zustand
- Tailwind CSS
- Framer Motion
